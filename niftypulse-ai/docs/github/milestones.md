# GitHub Milestones for NiftyPulse AI

## Milestone 1 — Project Foundation
- Description: Repository scaffolding, contribution workflows, CI skeleton, auth and baseline API/frontend setup, plus initial database schema and migration system.
- Suggested due date offset: Week 1-2

## Milestone 2 — Data Pipeline
- Description: Build reliable ingestion pipelines for Nifty 200 constituents, OHLCV, delivery volume, promoter holdings, and cashflow with validation and orchestration.
- Suggested due date offset: Week 3-4

## Milestone 3 — Signal Engine
- Description: Implement and test all EOD signal calculators, aggregation logic, persistence, and API exposure for signal analytics.
- Suggested due date offset: Week 5-6

## Milestone 4 — ML & Pattern Detection
- Description: Establish supervised ML experimentation, MLflow lifecycle, drift reporting, and rule-based chart pattern detection modules.
- Suggested due date offset: Week 7-8

## Milestone 5 — Frontend & Deployment
- Description: Build full React dashboard experience, connect all APIs, enforce compliance UX, and deploy via Netlify with CI/CD automation.
- Suggested due date offset: Week 9-10
